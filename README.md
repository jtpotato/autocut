# Installation
Simple. Fast.
```
npm i -g https://github.com/jtpotato/autocut/releases/download/v1.0.4/autocut-1.0.4.tgz
cd <directory where the videos are>
autocut
```
(Assumes `Node.js`, `npm`, `Python`, `pip` and `auto-editor` are installed.)
