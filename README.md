# Installation
Simple. Fast.
```
git clone https://github.com/jtpotato/autocut/
cd autocut
npm i -g .
cd <directory where the videos are>
autocut
```
(Assumes `Node.js`, `npm`, `Python`, `pip` and `auto-editor` are installed.)
